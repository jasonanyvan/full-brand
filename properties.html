Construction and clear space

When creating property specific lockups, remember to always write out the property name in the primary typeface, in capital case with 0% letter spacing and in a light font weight. If and when written out it, the property name should be stylized as AnyVan Property name - e.g. AnyVan Business, AnyVan Cars, AnyVan Storage

Minimum size

When used against white, the minimum size of the stacked version of the lock-up should be 60px, but if used against a constrasting colour, legibility is not compromised when the minimum size is reduced to 40px - as demonstrated below

Offerings

When dealing with property specific offerings, to avoid the use of overbranding, employ alternative branding elements to tie the offering back - namely colour or visual cues associated with that property. Never use property specific logos along side an offering lockup, instead use the primary logo and the offering name seperated by a pipe - as demonstrated below
